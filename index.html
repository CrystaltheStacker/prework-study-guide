Prework Study Guide

HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • HTML elements are made up of a start tag, content, and an end tag.
  • HTML elements can be nested inside of each other.
  • HTML elements can have attributes that provide additional information about the element.
  • HTML elements can be self-closing.
  • HTML elements can be block or inline.
  • HTML elements can be hidden from the user.
  • HTML elements can be used to group content together.
  • HTML elements can be used to create a list.
  • HTML elements can be used to create a table.
  • HTML elements can be used to create a form.
  • HTML elements can be used to create a link.
  • HTML elements can be used to create an image.
  • HTML elements can be used to create a video.
  • HTML elements can be used to create an audio file.

CSS

  • CSS elements is short for Cascading Style Sheets.
  • CSS is used to format the layout of a webpage.
  • CSS elements control the color, font and size of the text.
  • CSS elements control the color and size of the background, borders and margins.
  • CSS elements control the spacing between elements, how elements are positioned and laid out.
  • CSS elements can be used to style HTML elements
  • CSS elements can be used to style text,links and forms.
  • CSS elements can be used to style lists and tables.
  • CSS elements can be used to style images, videos and audio files.

Git

  • git status: checks what branch we are currently on
  • git add: adds files to the staging area
  • git commit: commits files to the local repository
  • git push: pushes files to the remote repository
  • git pull: pulls files from the remote repository
  • git checkout: switches between branches
  • git branch: creates a new branch
  • git merge: merges two branches together
  • git clone: clones a remote repository to your local machine
  • git remote: connects a local repository to a remote repository
  • git log: shows a list of commits
  • git reset: resets the staging area
  • git revert: reverts a commit
  • git rebase: rewrites the commit history
  • git tag: creates a tag for a commit
  • git fetch: fetches files from the remote repository

JavaScript

  • JavaScript adds interactivity to a website.
  • JavaScript can be used to create a popup box.
  • JavaScript can be used to create a function.
  • JavaScript can be used to create a variable.
  • JavaScript can be used to create an array.
  • JavaScript can be used to create an object.
  • JavaScript can be used to create a loop.
